# VRML2STL
VRML File Convert to STL File for python script

# Dependencies
- blender ( >= Blender 2.82 )
- python

# Usage
Conver WRL to STL
```
blender --background --python wrl2stl.py -- <input_folder> <output_folder>
```

Conver dae to STL
```
blender --background --python dae2stl.py -- <input_folder> <output_folder>
```
